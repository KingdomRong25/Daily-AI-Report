📌 TOP 10 重点资讯

🔴 S 级 01

Google Cloud Next '26 开幕：TPU v8 双芯片架构正式发布，Anthropic 承诺多吉瓦级采购

Google 在拉斯维加斯 Cloud Next 大会（4/22-24）发布第八代 TPU，首次将训练（TPU 8t）和推理（TPU 8i）分离为独立芯片。训练性能提升 2.8 倍，推理性能提升 80%，每美元性价比提升 80%。推理芯片 TPU 8i 每颗集成 384MB SRAM（前代3倍），专为低延迟、高并发 Agent 场景设计。更关键的是：Anthropic 已确认承诺多吉瓦级 TPU 采购规模，Broadcom 同意扩大与 Google/Anthropic 的芯片合作。这是 Google 向 NVIDIA 霸权发起的最有力冲击。

值得关注：推理芯片专业化 + Anthropic 背书 = NVIDIA 单极时代真正出现裂痕；对整个行业的推理成本下降有直接推动。
🔴 S 级 02

SpaceX 获得以 $600 亿收购 Cursor 的期权，AI 编程赛道估值天花板被彻底刷新

SpaceX（已合并 xAI）宣布获得以 600 亿美元估值收购 AI 编程工具 Cursor 的权利，或选择支付 100 亿美元购买双方联合开发成果。Cursor 将获准在 xAI 超级计算机上训练其"Composer"模型，摆脱对 OpenAI/Anthropic 的模型依赖。Cursor 目前正以超 500 亿估值融资 20 亿（a16z 领投）。此交易发生在 Musk 诉 Altman 案开庭前一周——OpenAI 曾是 Cursor 早期投资者。

值得关注：AI Coding 工具从"创业赛道"跃升为"巨头战略资产"，$600亿估值重新定义了开发工具的价值上限。
🟠 A 级 03

月之暗面开源 Kimi K2.6：编程能力对标 GPT-5.4，300 Agent 并发协作

月之暗面于 4/20 发布并开源旗舰模型 Kimi K2.6，在 SWE-Bench Pro 等编程基准上超越 GPT-5.4 和 Claude Opus 4.6。核心亮点：万亿参数、256K 上下文窗口、支持 300 个智能体并发调度、12 小时连续编程能力。API 定价较竞品上涨 58%，显示商业化信心。一周内连续迎来 Claude Opus 4.7 和 Qwen3.6-Max-Preview 竞品发布，国产大模型"闪电战"进入白热化阶段。

值得关注：国产开源模型首次在编程基准全面对标美系顶尖闭源模型，Agent 并发能力成为新竞争维度。
🟠 A 级 04

DeepSeek V4 采用华为昇腾芯片训练，阿里字节配合适配

据路透社独家报道，DeepSeek 最新一代 V4 模型将全程运行在华为昇腾芯片上，阿里、字节跳动等同步进行生态适配。此举标志着国产 AI 算力栈从"可用"走向"规模化实战"，在中美算力博弈背景下具有标志性意义。V4 定位为万亿参数级开源旗舰，编程与数学能力为核心卖点。

值得关注：国产算力全栈替代的关键验证节点，若成功将大幅降低中国 AI 企业对 NVIDIA 的依赖。
🟠 A 级 05

OpenAI 计划向 Cerebras 投资 $200+ 亿构建自有 AI 芯片基础设施

OpenAI 正式宣布计划向专用芯片公司 Cerebras 投资超过 200 亿美元换取股权，旨在构建超越现有供应商的 AI 基础设施。与此同时，OpenAI Codex 周活突破 300 万，新增 macOS 桌面自动化、90+ 插件集成；还发布了生物医学模型 GPT-Rosalind 和网络安全模型 GPT-5.4-Cyber。但高管动荡持续：Sora 负责人 Bill Peebles 和科学团队负责人 Kevin Weil 近期相继离职。

值得关注：$200亿芯片投资 + Codex 企业扩张 + 高管出走 = OpenAI 正在经历"战时模式"下的快速转型与阵痛。
🟡 B 级 06

Anthropic 安全风波：Mythos 网络工具遭未授权访问，Claude Code Routines 上线

据报道一个未授权团体获得了访问 Anthropic 独家网络工具 Mythos 的权限，引发 AI 安全与防御工具泄露的广泛担忧。产品方面，Claude Code 桌面应用重新设计，新增侧边栏多会话管理和 Routines（例程）功能，支持自动化 PR 审查和部署验证。

值得关注：Mythos 作为 Anthropic 的核心安全/攻防工具被入侵，安全公司的安全工具自身安全性受质疑。
